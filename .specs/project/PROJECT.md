# Campanhas Tipi da Serra

**Vision:** Explorar o banco de dados da Tipi da Serra, mapear relacionamentos entre tabelas e construir dashboards para criação e gestão de campanhas.

**For:** Equipe Tipi da Serra / OK IA (gestão de campanhas e dados)

**Solves:** Falta de visibilidade sobre o modelo de dados e necessidade de dashboards para campanhas baseadas em dados reais.

## Goals

- Mapear schema e relacionamentos do banco de dados existente (documentação e diagrama).
- Entender o domínio de campanhas no contexto da Tipi da Serra (entidades, regras, fluxos).
- Construir dashboards que permitam explorar dados e apoiar decisões de campanhas.

## Scope

**v1 inclui:**

- Exploração e documentação do banco (tabelas, relacionamentos, convenções).
- Modelo de domínio de “campanhas” no contexto Tipi da Serra.
- Primeiro dashboard funcional (métricas básicas / exploração de dados).

**Explicitamente fora do escopo (por enquanto):**

- Migração ou alteração de schema em produção sem aprovação.
- Múltiplos dashboards avançados antes de validar o primeiro.
